# MULTIFUNCTION-INSTRUMENTATION-SYSTEM-DAQ-
Repository Name: multifunction-instrumentation-daq Repository Description: Raspberry Pi 4-layer DAQ system integrating 6 instruments: Voltmeter, Ohmmeter, DC Reference, Square Wave Generator, Sine Wave Generator, and Frequency Measurer. Features optimized SPI-driven SAR ADC routines and a ±12V dual-rail push-pull supply.
